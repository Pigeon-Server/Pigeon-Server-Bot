# Pigeon Server Bot
A mirai plugin for remotely executing minecraft commands.

Pigeon Server · 气人姬部分功能实现


## 已实现功能
- 白名单申请
- 多服务器状态查询
- 分群问答
- 版本更新提示
## 待实现功能
- 多账户
- 多服务器
- 开黑啦同步
- 群文件自动分类
- 测试服权限申请
- 假人登记/自动踢出未登记假人
